Getting Started

Welcome to Phish Me Not! This tool allows you to create, launch, and analyze phishing simulations to enhance security awareness.

Follow these steps to begin:

  1. Install the tool and ensure all dependencies are set up.
  2. Configure your SMTP settings under the Settings page.
  3. Create your first phishing campaign from the Campaigns page.

Creating a Phishing Campaign

To create a phishing campaign:

  1. Go to the Campaigns page.
  2. Select a phishing template (e.g., Amazon, Microsoft, Twitch, etc.).
  3. Fill in campaign details such as sender email, target email, and email body.
  4. Generate and embed the phishing link inside your email body.
  5. Click Send to launch the campaign.

Configuring SMTP

To send phishing emails, you must configure SMTP settings:

  1. Go to the Settings page.
  2. Enter SMTP server details, including:
    • Server Address: e.g., smtp.gmail.com
    • Port: 587 for TLS, 465 for SSL
    • Encryption: SSL / TLS / None

Managing Results

After launching campaigns, you can analyze the results under the Results page.

Available data:

  • Total campaigns launched
  • Total credentials collected
  • Template used
  • Target
  • Captured credentials (emails/usernames and passwords)

Troubleshooting

Common issues and solutions:

  • Email Not Sending? Ensure SMTP credentials are correct.
  • Phishing Page Not Loading? Check if the template is hosted correctly.

FAQs

  • Q: Can I add my own phishing templates?
    A: Currently, templates are preloaded and cannot be modified.
  • Q: How can I analyze captured credentials?
    A: View all captured credentials under the Results page.
