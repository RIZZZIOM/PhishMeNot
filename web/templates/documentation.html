Getting Started

Welcome to PHISH ME NOT! Follow these steps to set up and get started:

  1. Install the tool.
  2. Configure SMTP settings on the Settings page.
  3. Create your first phishing campaign.

Creating a Campaign

Steps to create a new phishing campaign:

  • Go to the Campaigns page.
  • Select a template and fill in campaign details.
  • Add target email addresses and launch the campaign.

Configuring SMTP

To send phishing emails, you need to configure SMTP settings:

  1. Go to the Settings page.
  2. Enter your SMTP server details, including server address, port, username, and password.
  3. Test the connection to ensure it's working.

Managing Results

Analyze your campaigns on the Results page:

  • View the number of emails sent, opened, and clicked.
  • Check captured credentials and export results if needed.

Troubleshooting

Common issues and their solutions:

  • Ensure SMTP settings are correct if emails fail to send.
  • Check the internet connection for connectivity issues.
  • Contact support for unresolved issues.

FAQs

Answers to frequently asked questions:

  • Q: Can I add custom templates? A: Templates are preloaded and cannot be modified.
  • Q: How do I test my SMTP settings? A: Use the "Test Connection" button on the Settings page.
